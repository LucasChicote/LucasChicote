## 📌 Sobre mim
- Tenho 20 anos e estou cursando Análise e Desenvolvimento de Sistemas na FIAP. Amo tecnologia e estou focado em desenvolvimento Back-End, Banco de Dados e um pouco de Front-End.
- Meu objetivo é construir aplicações robustas, escaláveis e bem estruturadas, com domínio sólido em:
- ⚙️ Desenvolvimento Back-End com Java
- 🗄️ Banco de Dados relacionais com SQL / PL/SQL
- 🔌 Arquitetura e desenvolvimento de APIs REST
- 🐳 Containerização com Docker
- ☁️ Cloud e DevOps com Azure
- 🐍 Automação e scripts com Python
- 🎨 Noções de Front-End com TypeScript / React / Angular
